## Short Description (132 chars max)

Import LinkedIn profiles directly into Winnow for candidate sourcing, matching, and hiring pipeline management.

## Detailed Description

Winnow LinkedIn Sourcing lets recruiters and hiring managers import candidate
profiles from LinkedIn directly into the Winnow hiring platform.

**Features:**
- One-click extraction of LinkedIn profile data (name, headline, experience,
  education, skills)
- Secure OAuth sign-in with your Winnow account (LinkedIn, Google, GitHub,
  Microsoft, or Apple)
- Preview extracted data before saving
- Tag candidates to specific job openings
- Duplicate detection prevents re-importing existing candidates

**How it works:**
1. Sign in with your Winnow employer account
2. Navigate to any LinkedIn profile page
3. Click the Winnow extension icon
4. Review the extracted profile data
5. Save to Winnow, optionally tagging to a job

**Requirements:**
- A Winnow account with employer access (sign up at winnowcc.ai)
- Chrome 88 or later
